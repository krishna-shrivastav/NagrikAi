# 🚀 NagrikAI

AI-powered platform to simplify access to government schemes and public services.

## 🔍 Problem
Citizens face difficulty accessing government schemes due to:
- Complex procedures
- Lack of awareness
- Language barriers
- Errors in form filling

## 💡 Solution
NagrikAI provides:
- AI-based scheme recommendations
- Eligibility checking
- Smart form autofill
- Multilingual guidance
- Secure redirection to official portals

## 🛠 Tech Stack
- Frontend: React / Next.js
- Backend: Node.js / Express
- Database: MongoDB
- AI: OpenAI / LLM APIs

## ⚙️ Setup Instructions

### 1. Clone Repo
```bash
git clone https://github.com/your-username/NagrikAI.git
cd NagrikAI
```
### 2.Backend Setup
```bash
cd backend
npm install
npm start
```
###3.Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
